Fonctions et événements JavaScript

un JavaScript function est un bloc de code JavaScript, qui peut être exécuté lorsqu'il est « appelé »

une fonction peut être appelée lorsqu’un événement se produit, comme lorsque l’utilisateur clique sur un bouton.

JavaScript externe

Les script peuvent également être placé dans des fichier externe.

Les scripts externes sont pratiques lorsque le même code est utilisé dans de nombreuses pages Web différentes.

Pour utiliser un script externe, placez le nom du fichier de script dans l' srcattribut (source) d'une <script> balise

Example: <script src="file.js"></script>

Java Script Output

Possibilité d'affichage du Javascript

Js peut afficher des donnée de différentes manières

  • Ecrire dans un element HTML, en utilisant innerHTML ou innerText.
  • Ecrire dans la sortie HTML à l'aide de document.write()
  • Ecrire dans une boite d'alerte, en utilisant window.alert().
  • Ecriture dans la console du navigateur, en utilisant console.log().

Utilisation de innerHTML

Pour accéder à un élément HTML, vous pouvez utiliser la document.getElementById(id) méthode

utilisez l'id attribut pour identifier l'élément HTML.

Utiliser ensuite la innerHTML pour modifier le contenue HTML de l'element HTML.

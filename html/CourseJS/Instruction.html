Programme JavaScript

un programme informatique est une liste d'instruction à executer par un ordinateur.

dans un langage de programmation, ces instruction de programmation sont appelées instruction.

Instruction JavaScript

Les instructions JavaScript sont composées de :

  • valeurs
  • opérateurs
  • expressions
  • mots-clés
  • commentaires

JS Syntaxe

Valeurs JS

la Syntaxe JS définit deux type de valeurs:

  • valeurs fixes
  • valeurs variables

Variables JS

les variables sont utilisées pour stocker des valeurs de données. JavaScript utilise les mots-clés var, let et const pour déclarer des variables

Un signe égal (=) est utilisé pour attribuer des valeurs aux variables.

let JS

  • Les variables déclarées avec let ont une portée de bloc
  • Les variables déclarées avec let doivent être déclarées avant utilisation
  • Les variables déclarées avec let ne peuvent pas être redéclarées dans la même portée

Les variables déclarées à l'intérieur d'un bloc { } ne sont pas accessibles depuis l'extérieur du bloc

Portée mondiale

Les variables déclarées avec var ont toujours une portée global

Les variables déclarées avec le var mot-clé NE PEUVENT PAS avoir une portée de bloc :

let et const ont une portée bloc

let et const ne peut pas être redéclaré

let et const doit être déclaré avant utilisation.

let et const ne se lie pas à this

let et const sont pas hissé

Les variables définies avec var sont hissées au sommet et peuvent être initialisées à tout moment

signification: Vous pouvez utiliser la variable avant qu’elle ne soit déclarée

Système de fichier

Comment utiliser les applications de gestion de fichiers ?

Une application de gestion de fichiers permet de stocker, d'organiser et de modifier facilement vos fichiers sur votre ordinateur

Une façon d'organiser vos fichiers est d'utiliser des étiquettes

Pour rechercher des éléments étiquetés, vous pouvez faire défiler la barre latérale jusqu'en bas et cliquer sur l'étiquette colorée correspondante, ou vous pouvez rechercher l'étiquette par nom de couleur dans le champ de recherche

Une autre façon d'organiser vos fichiers est d'utiliser des dossiers intelligents

Ces dossiers sont des collections de fichiers automatiquement organisées selon des critères définis, tels que le type de fichier, la date ou des mots-clés.

Quelles sont les meilleures pratiques pour nommer les fichiers des applications Web ?

utiliser des noms descriptif comme produit-supermarche.html, le contenu du fichier sera clair

Il est donc préférable de n’utiliser que des lettres et des tirets dans les noms de vos fichiers.

Quelles sont les meilleures pratiques pour l’organisation des fichiers et des dossiers dans les applications Web ?

Lors du développement d'applications web, il est important de penser à l'organisation de vos fichiers et dossiers. Cela permettra de maintenir votre code organisé et d'en faciliter la maintenance.

  • Dans le répertoire racine, le premier répertoire affiché est le assets répertoire. Ce terme assets désigne tous les fichiers utilisés dans le projet, tels que les images, les polices ou autres ressources.

    • Dans cet exemple, il y a deux répertoires à l'intérieur du assets répertoire appelé images et fonts.

  • Le répertoire suivant est le css répertoire
  • Sous le css répertoire se trouvent les fichiers HTML.
  • README.md ces fichiers sont généralement utilisés pour fournir des informations utiles sur le projet, telles que ce qu'il fait, comment l'utiliser et tout autre détail pertinent.

    • .md extension signifie Markdown, qui est un langage de balisage léger souvent utilisé pour écrire de la documentation.

Quels sont les types de fichiers courants utilisés dans les applications Web ?

Les fichiers sont classés selon leur contenu et leur structure. Le type de fichier détermine comment un fichier est ouvert et traité par les programmes informatiques

Nous identifions les types de fichiers par leur extension. Une extension de fichier est un suffixe de trois ou quatre lettres à la fin du nom du fichier

voici les différente extension pour les images

  • Le format JPEG(Joint Photographic Experts Group) est réputé pour son efficacité en termes de compression. Il est idéal pour les photographies et les images. Cependant, cette compression entraîne des pertes, ce qui signifie qu'une partie de la qualité de l'image est perdue pour réduire la taille du fichier
  • Le format PNG (Portable Network Graphics) préserve la qualité de l'image, même après compression. Cela permet d'obtenir des fichiers plus volumineux. Le format PNG prend en charge la transparence et est idéal pour les images aux contours nets, comme les logos et les icônes.
  • Le format d'image GIF (Graphics Interchange Format) prend en charge l'animation et la transparence, mais sa palette de couleurs est limitée
  • SVG ( Scalable Vector Graphics) C'est un format d'image vectorielle. Ces images peuvent être redimensionnées à volonté sans perte de qualité

Voici quelques extension pour les audio et video

  • MP3 un format audio réputé pour son excellente compression. Il s'agit d'un format avec perte ; certaines données audio sont donc perdues lors de la compression, qui vise à réduire la taille des fichiers.
  • MP4 est l'un des formats vidéo les plus courants. Il offre une bonne compression et prend en charge de nombreux codecs audio et vidéo, sous-titres et métadonnées.
  • MOV le format de fichier multimédia QuickTime développé par Apple, est principalement associé à QuickTime Player.

D'autres types de fichiers audio et vidéo populaires sont WAV, un format audio sans perte qui conserve la qualité originale de l'audio, et WebM, un format vidéo open source de haute qualité.

Pour personaliser les polices de vos application vous aller utilisez des format de polices voic les trois plus utilisé :

  • TTF (TrueType Font) est un format de police compatible avec différents systèmes d'exploitation.
  • WOFF (Web Open Font Format) est un format de police moderne, spécialement conçu pour le développement web, Ils peuvent également stocker des métadonnées, notamment des informations de licence.
  • WOFF2 est le successeur de WOFF avec une compression et des performances encore plus efficaces.

La documentation dans les applications web on a le README qui contient des informations sur l'application, comme son utilisation, son installation, sa licence et les modalités de contribution.

Ils sont généralement rédigés dans un format de fichier appelé Markdown. Avec Markdown, vous pouvez créer des documents structurés avec des titres, des sous-titres, des liens, des images, des listes, etc.

Comprendre les bases de l'informatique, d'Internet et des outils de développement

  • Carte mère elle contient toute la mémoire, les connecteurs et les disques durs nécessaires au fonctionnement de l'ordinateur. Elle sert de circuit imprimé principal.
  • Unité centrale de traitement (CPU) un processeur chargé d'exécuter des instructions et d'effectuer des calculs.
  • Mémoire vive (RAM) un emplacement de stockage temporaire pour le processeur de l'ordinateur.
  • Disque dur (HDD) un emplacement de stockage permanent utilisé pour stocker des données même lorsque l'ordinateur est éteint.
  • Solid State Drive (SSD) mémoire flash non volatile et peut être utilisée à la place d'un disque dur.
  • Bloc d'alimentation (PSU) chargé de convertir l'électricité de la prise murale en une forme utilisable par l'ordinateur.
  • Unité de traitement graphique (GPU) responsable du rendu des visuels sur l'écran de l'ordinateur.
  • Différents types de fournisseurs d'accès à Internet Un fournisseur d'accès à Internet (FAI) est une entreprise qui fournit un accès à Internet. Il existe différents types de FAI, notamment l'accès commuté, l'ADSL, le câble, la fibre optique et le satellite.
  • Moyens sûrs de se connecter à votre ordinateur Voici des exemples de moyens sûrs de se connecter à votre ordinateur : l'utilisation d'un mot de passe fort, l'activation de l'authentification à deux facteurs et l'utilisation d'un gestionnaire de mots de passe
  • Environnement de développement intégré (IDE) un outil qui aide les développeurs à écrire, tester et déboguer du code de manière efficace.
  • Éditeur de code un outil que les développeurs utilisent pour écrire et déboguer du code
  • Git un système de contrôle de version populaire qui permet aux développeurs de suivre les modifications de leur code et de collaborer avec d'autres.
  • Services d'hébergement cloud pour dépôts un dépôt est un emplacement de stockage pour les fichiers de projet et l'historique des versions. Parmi les services d'hébergement cloud les plus populaires, on trouve GitHub, GitLab et Bitbucket.
  • Gestionnaires de paquets outils qui simplifient l'ajout, la mise à jour et la suppression de bibliothèques et de dépendances de projet. Exemples : npm, pip et Maven.
  • Bibliothèques et frameworks de test les tests sont effectués dans le logiciel pour garantir que le code fonctionne comme prévu. Jest, PHPUnit et JUnit sont des exemples de bibliothèques et de frameworks de test.

Travailler avec des fichiers, des systèmes de fichiers et des formats multimédias

  • Bonnes pratiques pour nommer les fichiers l est conseillé de nommer vos fichiers de manière à ce qu'ils soient faciles à comprendre et à gérer. Par exemple, « [.html] » about-us.htmlest un nom plus descriptif que page1.html« .html »
  • Répertoire racine répertoire de niveau supérieur dans un système de fichiers. Répertoire est un autre nom pour un dossier.
  • Markdown langage de balisage couramment utilisé pour la documentation et READMEles fichiers. Un READMEfichier contient des informations sur un projet, comme son installation et son utilisation.
  • index.html représente la page par défaut qui s'affiche lorsqu'un utilisateur visite un site Web.

Naviguer efficacement sur le Web

  • Qu'est-ce qu'un navigateur Web ? une application logicielle qui permet aux utilisateurs d'accéder et de visualiser des sites Web sur Internet.
  • Qu'est-ce qu'un moteur de recherche ? Un outil permettant aux utilisateurs de rechercher des informations sur Internet. Google, Bing et Yahoo en sont des exemples.
  • Navigateurs Web courants Quelques exemples de navigateurs Web courants incluent Google Chrome, Mozilla Firefox et Microsoft Edge.

CSS Flexbox

Flexbox est l'abréviation du module Flexible Box Layout

Flexbox est une méthode de mise en page permettant d'organiser des éléments en lignes ou en colonnes

Flexbox facilite la conception d'une structure de mise en page réactive et flexible, sans utiliser de flottement (float) ni de positionnement (position).

Module de positionnement

avant l'arrivé du module flexible box layout il exister quatre modes de mise en page :

  • block - pour les sections d'une page Web
  • Inline - pour le texte
  • Table - pour les données de tableau à deux dimensions
  • Positioned - pour la position explicite d'un élément

Composant flexbox

Une flexbox est toujours composée de

  • un conteneur Flex - l'élément parent (conteneur) <div>
  • Éléments flexibles - les éléments à l'intérieur du conteneur <div>

Un conteneur flexible avec trois éléments flexibles

Pour commencer à utiliser CSS Flexbox, vous devez d'abord définir un conteneur flexible.

Le conteneur flexible devient flexible en définissant la display propriété sur flex

Flex Container

Comme on l'a vu il s'agit d'un conteneur flexible avec des éléments flexibles à l'intérieur

le conteneur devient flexible en définissant display:flex

les proprièté CSS que nous utilisons pour le conteneur flex sont:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

Flex-direction

La flex-direction spécifie la direction d'affichage des éléments flexibles dans le conteneur flexible.

elle peut avoir une des valeurs suivante :

  • row
  • column
  • row-reverse
  • column-reverse

Flex-wrap

La flex-wrap spécifie si les éléments flexibles doivent être enveloppé ou non, s'il n'y a pas assez de place pour eux sur une ligne flexible

elle peut avoir une des valeurs suivante :

  • nowrap
  • wrap
  • wrap-reverse

Flex-flow

La flex-flow une propriété abrégée permettant de définir à la fois les propriétés flex-direction et flex-wrap

justify-content

La justify-content est utilisée pour aligner les éléments flexibles lorsqu'ils n'utilisent pas tout l'espace disponible sur l'axe principal (horizontalement).

elle peut avoir l'une des valeurs suivantes:

  • center
  • flex-start
  • flex-end
  • space-around
  • space-between
  • space-evenly

align-items

la align-items est utilisée pour aligner les élément flexible lorsqu'ils n'utilisent pas tout l'espace disponible sur l'axe transversal(verticalement )

elle peut avoir l'une des valeurs suivantes:

  • center
  • flex-start
  • flex-end
  • stretch
  • baseline
  • normal

align-content

la align-content est utilisée pour aligner les lignes flexibles

elle est similaire à align-items mais au lieu d'aligner les éléments flexibles, elle aligne les lignes flexibles

Elle peux avoir l'une des valeurs suivantes

  • center
  • stretch
  • flex-start
  • flex-end
  • space-around
  • space-between
  • space-evenly

Flex-items

Les propriéte que nous utilisons pour les élément flexible sont

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self

order

spécifie l'ordre des éléments flexibles à l'intérieur du conteneur flexible.

Exemple : style:"order 3" il sera possitioner en 3ème

flex-grow

spécifie dans quelle mesure un élément flexible va croître par rapport au reste des éléments flexibles fonctionne un peu comme colspan

Faites en sorte que le troisième élément flexible se développe huit fois plus vite que les autres éléments flexibles :

Exemple : style="flex-grow:8"

flex-shrink

spécifie dans quelle mesure un élément flexible va rétrécir par rapport au reste des éléments flexibles.

flex-basis

la flex-basis spécifie la longueur initiale d'un élément flexible.

Flex

est une propriété abrégée pour les propriétés flex-grow, flex-shrink, flex-basis

align-self

spécifie l'alignement de l'élément sélectionné à l'intérieur du conteneur flexible.

remplace l'alignement par défaut défini par la align-items propriété du conteneur.

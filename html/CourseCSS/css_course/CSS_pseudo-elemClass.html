CSS Pseudo Classes & Elements

Pseudo-classes

Est utilisée pour définir un état spécial d'un élément.

  • Styliser un élément lorsqu'un utilisateur déplace la souris dessus
  • Styliser différemment les liens visités et non visités
  • Appliquer un style à un élement lorsqu'il est mis en avant
  • Style des élements de formulaire valides/Invalides/Obligatoires/Facultatifs

Syntax Pseudo-classes : Selector:pseudo-class

comme on l'a vu précedemment avec les lien (:link, :visited, :hover, :active) sont des Pseudo-classes.

On peur aussi les utiliser sur d'autre balise que la balise d'ancrage (a)

Pseudo-classe : first-child

La :first-child pseudo-classe correspond à un élément spécifié qui est le premier enfant d'un autre élement

Dans l'exemple suivant, le sélecteur correspond à tout élément <p> qui est le premier enfant de n'importe quel élément :

p:first-child

Faire correspondre le premier élément <i> dans tous les éléments

le sélecteur correspond au premier élément <i> dans tous les éléments <p> : p i:first-child

Faire correspondre tous les éléments <i> dans tous les premiers éléments enfants

e sélecteur correspond à tous les éléments <i> dans les éléments <p> qui sont le premier enfant d'un autre élément : p:first-child i

Pseudo-classe: lang

La :lang pseudo-class permet de définir des règles spéciales pour différente langues.

Pseudo-Elements

Un Pseudo-élément CSS est utilisé pour styliser des partie spécifique d'un élément.

  • styliser la première lettre ou ligne d'un élément
  • Insérer du contenu avant ou après un élément
  • Styliser les marqueurs des éléments de la liste
  • Styliser la zone d'affichage derrière une boîte de dialogue

Syntax : selector::pseudo-element {}

pseudo-element ::first-line

Le ::first-line est utilisé pour ajouter du style spécial a la première ligne d'un texte.

Remarque le ::first-line pseudo-élément ne peut être appliqué qu'aux éléments de niveau bloc.

pseudo-element ::first-letter

The ::first-letter est utilisé pour ajouter un style spécial à la première lettre d'un texte

Remarque : le ::first-letter pseudo-élément ne peut être appliqué qu'aux éléments de niveau bloc

pseudo-element ::before

Le ::before est utilisé pour insérer du contenue avant le contenu d'un élément.

pseudo-element ::after

le ::after est utilisé pour inserer du contenu après le contenu d'un élément.

pseudo-element ::marker

Le ::marker sélectionne les marqueurs des éléments de la liste.

pseudo-element :: selection

Le ::selection correspond à la partie d'un élément selectionnée par un utilisateur les proprièté qui peuvent lui être appliquer

  • color
  • background
  • cursor
  • outline

Media Query

La @media permet de définir différente règle de style pour différent type de média(appareil)

Les requêtes multimédias peuvent être utilisées pour vérifier de nombreuses choses, telles que :

  • largeur et hauteur de la fenêtre d'affichage
  • orientation de la fenêtre d'affichage (paysage ou portrait)
  • résolution

Type de média CSS
Value Description
all Used for all media type devices
print Used for print preview mode
screen Used for computer screens, tablets, smart-phones etc.

Qui sont souvent utilisé

Fonctionnalité multimédias commune CSS
Value Description
orientation Orientation of the viewport. Landscape or portrait
max-height Maximum height of the viewport
min-height Minimum height of the viewport
height Height of the viewport (including scrollbar)
max-width Maximum width of the viewport
min-width Minimum width of the viewport
width Width of the viewport (including scrollbar)

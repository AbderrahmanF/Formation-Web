CSS Combinators

A CSS selector can contain more than one simple selector. Between the simple selector, we can include a combinator.

There are four different Combinators in CSS:

  • Descendant combinator (space)
  • child combinator (>)
  • Next siblling combinator (+)
  • Subsequent-sibling combinator (~)

Descendant Combinator

Le combinateur Descendant correspond à tous les éléments qui sont les Descendants d'un élément spécifié

Par exemple une balise <p> à l'intérieur d'une balise <div> la balise p descend de div

la syntax est div p {}

Child Combinator (>)

Le combinateur enfant sélectionne tous les élements qui sont les enfant d'un éléments spécifié

exemple : div > p ce selecteur sélectionne tout les element p enfant d'un element div si par exemple j'ai une à l'intérieur du div une balise section et qu'a l'interieur de cette balise j'ai un div ce selecteur ne prendra pas en compte ce p

Next siblling Combinator (+)

Le combinateur frère suivant est utilisé pour sélectionner un élement qui se trouve directement après un autre élément spécifique.

Les éléments frère doivent avoir le même élément parent, et " adjacent " immédiatement suivant.

Exemple: div + p {} selectionne le premier élement <p> placé immédiatement après les élement <div>

Subsequent-sibling combinator (~)

Le combinateur de frère et soeur suivant sélectionne tous les élements qui sont frère et soeur suivant d'un élement spécifié

cible tous les <p> qui suivent le <div> et qui partagent le même parent.

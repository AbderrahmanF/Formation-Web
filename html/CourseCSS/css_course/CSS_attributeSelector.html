Attribute Selector

Il est possible de styliser des éléments HTML qui ont des attributs ou des valeurs d'attributs spécifique.

Selecteur CSS [attribut]

Le [attribute] sélecteur est utilisé pour sélectionner des éléments avec un attribut spécifié

example : a[target]

ce sont tout les attributs a possedant l'attribut target

Selecteur CSS [attribut="value"]

Le [attribute="value"] sélecteur est utilisé pour sélectionner des éléments avec un attribut et une valeur spécifiés

example: a[target="_blank"]

Selecteur CSS [attribut~="valeur"]

Le [attribute~="value"] est utilisé pour sélectionner des éléments avec une valeur d'attribut contenant un mot spécifié.

Exemple slectionnant tous les élément avec un attribut title qui contient une liste de mot séparés dont l'un contient fleur

[title~="fleur"]

Selecteur CSS [attribut|="valeur"]

Le [attribut|="valeur"] est utilisé pour sélectionner les éléments avec l'attribut spécifié dont la valeur peut être exactement ka valeur spécifié, ou la valeur spécifié suivi d'un tiret (-)

Example : [class|="top"]

Sélecteur CSS [attribut^="valeur"]

Le [attribut^="valeur"] est utilisé pour sélectionner les éléments avec l'attribut spécifié, dont la valeur commence par la valeur spécifiée.

Example : [class^="top"]

Sélecteur CSS [attribute$="value"]

Le [attribute$="value"] est utilisé pour sélectionner les éléments dont la valeur d'attribut se termine par une valeur spécifiée.

Example : [class$="test"]

Sélecteur CSS [attribut*="valeur"]

Le [attribute*="value"] est utilisé pour sélectionner les éléments dont la valeur d'attribut contient une valeur spécifiée.

Example: [class*="te"]

Choosing the right font for your website is important!

Generic Font families

In CSS there are five generic font families:

  1. serif
  2. sans-serif
  3. Monospace
  4. cursive
  5. fantasy

In CSS, we use the font-family property to specify the font of a text

CSS Font Style

The font-style property is mostly used to specify italic text.

This property has three values:

  • normal
  • italic
  • oblique

Font weight

The font-weight property specifies the weight of a font

Font Variant

The font-variant property specifies whether or not a text should be displayed in a small-caps font

the value is : normal or small-caps

Font Pairing Rules

Voici quelques règles de base pour créer de superbes associations de polices :

  1. Complément : Il est toujours prudent de trouver des associations de polices qui se complètent. Une bonne combinaison de polices doit être harmonieuse, sans être trop similaire ou trop différente.
  2. Utiliser des superfamilles de polices : Une superfamille de polices est un ensemble de polices conçues pour fonctionner ensemble
  3. Le contraste est roi : Deux polices trop similaires entrent souvent en conflit. Cependant, les contrastes, bien utilisés, font ressortir le meilleur de chaque police.
  4. Choisissez un seul patron : Une police doit être la police principale. Cela établit une hiérarchie entre les polices de votre page

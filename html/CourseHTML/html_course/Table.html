HTML Table

Define an HTML Table

a table in HTML consists of table cells inside rows and columns

Table des balises concernant la création d'un tableau en HTML
Tag Description
<table> Defines a table
<th> Defines a header cell in a table
<tr> Defines a row in a table
<td> Defines a cell in a table
<caption> Defines a table caption the title of the table
<colgroup> Specifies a group of one or more columns in a table for formatting
<col> Specifies a group of one or more columns in a table for formatting
<col> Specifies column properties for each column within a <colgroup> element
<thead> groups the header content in a table
<tbody> Groups the body content in a table
<tfoot> Groups the footer content in a table

The <col> tag can use the span=2 attribute to apply styling to multiple columns at once. 

 Organisation d'un tableau :
            <thead>; <th> ... </thead>
            <tbody> ... </tbody> 
            <tfoot> ... </tfoot>

HTML Table - Colspan

HTML Table - Rowspan

To make a cell span over the multiple rows, use the rowspan attribute

Tableaux CSS

L'apparence d'un tableau HTML peut être grandement améliorée avec CSS

Full-width table

Pour gerer la taille du tableau

Collapse Table Borders

The border-collapse property sets whether the table borders should be collapsed into a single border

The width and height of a table are defined by the width and height properties.

Table Alignement

The text-align property sets the horizontal alignment of the content in <th> or <td>

The vertical-align property sets the vertical alignment (like top, bottom, or middle) of the content in <th> or <td>

CSS Table Style

use attribute padding to control the space between the border and the content in a table

horizontal Dividers

add the border-bottom property to <th> and <td>

Hoverable Table

Striped Tables

For zebra-striped tables, use the nth-child() selector and add a background-color to all even

CSS Responsive Table

A responsive table will display a horizontal scroll bar if the screen is too small to display the full content

Add a container element (like <div>) with overflow-x:auto around the <table> element to make it responsive

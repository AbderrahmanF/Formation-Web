Rappel des notions sur les tableaux en HTML

A quoi servent les tableaux HTML et a quoi ne devraient-ils pas servir ?

De nos jours les tableaux HTML sont moins utilisée qu'avant.

Les tableaux ont été l'un des premiers moyens d'affichage de donnée dans un navigateur dans les année 1990

  • table : utilisé pour créer un tableau HTML.
  • thead element : utilisé pour regrouper le contenu de l'en-tête dans un tableau HTML.
  • Élément Table Row (tr) : utilisé pour créer une ligne dans un tableau HTML.
  • Élément En-tête de tableau (th) : utilisé pour créer une cellule d'en-tête dans un tableau HTML.
  • tbody : utilisé pour regrouper le contenu du corps dans un tableau HTML.
  • Élément Table Data Cell (td) : utilisé pour créer une cellule de données dans un tableau HTML.
  • tfoot : utilisé pour regrouper le contenu du pied de page dans un tableau HTML.
  • caption élément : utilisé pour ajouter un titre à un tableau HTML.
  • colspan utilisé pour spécifier le nombre de colonnes sur lesquelles une cellule de tableau doit s'étendre.
  • rowspan utilisé pour spécifier le nombre de ligne sur lesquelles une cellule de tableau doit s'étendre.

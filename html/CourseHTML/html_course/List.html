HTML Liste de description

Que sont les liste de description ?

Les liste de description sont parfaite pour présenter des termes et des définitions dans un format organisé et facile à lire

example : 

            <dl>
                <dt>HTML</dt>
                <dd>HyperText Markup Language</dd>
                <dt>CSS</dt>
                <dd>Cascading Style Sheets</dd>
             </dl>

Dans cet exemple, <dl> est utilisé pour créer une liste de description, <dt> pour les termes et <dd> pour les définitions.

CSS List

Modifier le marqueur de la liste avec la propriété list-style-type

Ou pour definir une image comme marqueur on utilise list-style-image

Positionner les marqueurs d'éléments de liste

La list-style-position outside or inside définit si les puces seront a l'intérieur de l'élément de la liste ou non

Supprimer les paramètres par défaut

utiliser list-style-type:none Notez que la liste possède également une marge et un remplissage par défaut. Pour les supprimer, ajoutez «margin:0 et » padding:0à <ul> ou <ol>

raccourcis

Use list-style est une propriété abrégée

Example : list-style: square inside url("sqpurple.gif");

Lors de l'utilisation de la propriété abrégée, l'ordre des valeurs est :

  • list-style-type
  • list-style-position
  • list-style-image

HTML Quotation and Citation Elements

Vous devez faire preuve de discernement pour trouver le bon équilibre entre l’information et la présentation afin d’éviter d’encombrer le texte tout en étant clair et concis.

HTML <blockquote> for quotation

The HTML <blockquote> element defines a section that is quoted from another source. Browsers usually indent <blockquote≶ elements

The "cite" attribute in the <blockquote> tag specifies the source of the citation.

HTML <q> for Short Quotations

The HTML <q> tag defines a short quotation

Browsers normally insert quotation marks around the quotation

HTML <abbr> for abbreviations

The HTML <abbr> tag defines an abbreviation or an acronym, like "HTML", "CSS"

TIPS Use the global title attribute to show the description for the abbreviation/acronym when you mouse over the element

HTML <address≶ for contact information

The HTML <address> tag defines the contact information for the author/owner of a document or an article.

The contact information can be an email address, URL, Physical address, Phone number

the text in the address element usually renders in italic, and Browsers will always add a line break before and after the address

HTML <cite> for Work Title

The HTML <cite> tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, etc..)

Note: A person's name is not the title of a work.

HTML <bdo> for Bi-Directional Override

BDO stands for bi-directional Override

The HTML <bdo> tag is used to Override the current text direction

This tag is used with the dir attribute, which can take two values: "rtl" (right to left) or "ltr" (left to right).

Cours a modifier

le HTML "Citation en bloc" doit être utilisé pour représenter une section citée d'une autre source. Si la source de la citation possède une addresse, vous pouvez la citer avec cite attribut la valeur de cet attribut doit être une URL valide.

Bien que cet attribut ne modifie pas la présentation de la citation, il est très utile pour fournir aux lecteurs d'écran et aux moteurs de recherche plus d'informations sur la citation

HTML est le langage de balisage standard pour les pages Web.

Si vous souhaitez attribuer visuellement la source, vous pouvez ajouter un élément de citation, cite, à l'extérieur de l'élément de citation en bloc. Cet élément est différent de l' cite attribut. L'élément de citation est un élément HTML permettant de baliser le titre d'une œuvre référencée, comme un article de livre, une chanson, un film, un site web ou un article de recherche

HTML is the foundation of the web.

Spécification de la balise <address>

L'élément address de contact sert a représenter les coordonnées d'une section d'une page ou les informations de contact de l'auteur du document.

Est réserver au information de contact d'une personne

Lorsqu'il s'agit de crée les sections de contact de votre site Web, vous devez utiliser l'address éléments sémantique plutôt qu'un élément générique comme un div.

Example : 

    <address>
            <h2>Company Name</h2>
        <p>
            1234 Elm Street
            Springfield, IL 62701
            United States
        </p>
         <p>Phone: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
         <p>Email: <a href="mailto:contact@company.com">contact@company.com</a></p>
    </address>

un inconvénient de mailto est que les utilisateurs le perçoive comme du spam.

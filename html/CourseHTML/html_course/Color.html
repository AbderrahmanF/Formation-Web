HTML Colors

HTMl RGB and RGBA colors

An RGB color value represents RED, GREEN, BLUE light source

An RGBA color value is an extension of RGB with an alpha channel(opacity)

RGB Color Value

In HTML, a color can be specified as an RGB value, using this formula: rbg(red, green, blue)

Each parameter (red, green, blue) defines the intensity of the color with a value between 0 and 255

RGBA Color Values

RGBA color value is specified with rgba(red, green, blue, alpha(opacity))

The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparenta at all)

HTML HEX colors

A hexadecimal color is specified with #RRGGBB

HEX Color values

are hexadecimal value between 00 and ff (same as decimal 0-255)

HTML HSL and HSLA Colors

HTML stands for hue, saturation, and lightness

HTML color values are an extension of HSL with an Alpha channel(opacity)

HSl Color Values

In HTML, a color can be specified using hue, saturation and lightness (HSL) in the form: hsl(hue, saturation, lightness)

  • Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is blue and 240 is blue.
  • saturation is a percentage value. 0% means a shade of gray, and 100% is the full Color
  • Lightness is also a percentage value. 0% is black and 100% is white

Saturation can be described as the intensity of a color

  • 100% is pure color, no shade of gray
  • 50% is 50% of gray, but you can still see the color
  • 0% is completely gray; you can no longer see the color.

Lightness

The lightness of a color can be described as how much light you want to give the color, where 0% means no light (black), 50% means 50% light (neither dark nor light), and 100% means full lightness (white).

HSLA Color Values

HSLA color values is an extension of HSL color values, with an Alpha channel - which specifies the opacity for a color

An HSLA color value is specified with hsla(hue, saturation, lightness, opacity)

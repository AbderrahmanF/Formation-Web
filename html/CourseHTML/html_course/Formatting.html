HTML Text Formatting

HTML contains several elements for defining text with a special meaning

HTML Formatting Element

Formatting elements were designed to display special types of text

  • <b> - Bold text
  • <strong> - Important text
  • <i> - Italic text
  • <em> - Emphasized text
  • <matk> - Marked text
  • <small> - Smaller text
  • <del> or <s> - Deleted text
  • <ins> - Inserted text
  • <sub> - Subscript text
  • <sup> - Superscript text
  • <u> - surligner du text

HTML <b> and <strong> Elements

The HTML <b> element defines bold text, without any extra importance.

The HTML <strong> element defines text with strong importance. The content inside is typically displayed in bold

HTML <i> and <em> Elements

The HTML <i> element defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic

The HTML <em> element defines Emphasized text. The content inside is typically displayed in italic

HTML <small> Element

The HTML <small> element defines smaller text

HTML <mark> Element

The HTML <mark> element defines text that should be marked or highlighted

HTML <del> Element

The HTML <del> element defines text that has been deleted from a document. Browsers will usually strike a line through deleted text

HTML <ins> Element

The HTML <ins> element defines a text that has been inserted into a document. Browsers will usually underline inserted text

HTML <sub> Element

The HTML <sub> element defines subscript text. Subscript text appears half a character below the normal line, he can be used for the chemical formule, like H2O

HTML <sup> Element

The HTML <sup> element defines Superscript text appears half a character above the normal line 22=4

A quoi servent les éléments u, s, ruby et comment fonctionne t'il ?

l'élément d'annotation non articulé, ou u élément en abrégé est utilisé pour représenter un texte en ligne auquel une annotation non textuelle est appliqué

il u ne doit être utilisé que pour indiquer qu'un texte est étiqueté.

Elément barré, ou s élément en abrégé, doit être utilisé pour signaler que le texte n'est plus exact ou pertinent.

Cet s élément ne doit jamais être utilisé uniquement pour afficher les modifications apportées à un document. Dans ce cas, les éléments de texte supprimé (del) et inséré (ins) seraient plus appropriés.

Cet ruby élément représente un petit texte affiché au-dessus ou en dessous du texte principale.

Il est généralement utilisé pour indiquer la prononciation des caractères d'Asie de l'Est

Example :

<ruby> 明日 <rp> (</rp><rt>Ashita <rt><rp>)</rp> </ruby>

Resultat :

明日 (Ashita)

L'rp élément, ou élément de parenthèse de secours Ruby, est utilisé comme solution de secours pour les navigateurs ne prennant pas en charge l'affichage des annotations Ruby

L'rt élément, ou élément de texte Ruby, sert à indiquer le texte de l'annotation Ruby. Ce texte est généralement utilisé pour la prononciation ou les détails de traduction dans la typographie d'Asie de l'Est.

Bien que l'ruby élément puisse être utilisé pour d'autres types d'annotations, le cas d'utilisation le plus courant est pour la typographie d'Asie de l'Est.

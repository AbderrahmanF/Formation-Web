Rappel Notion en Semantique

L'importance du HTML sémantique

  • Hiérarchie structurelle des éléments de titre : il est important d'utiliser le bon élément de titre pour maintenir la hiérarchie structurelle du contenue
  • Elements HTML de présentation : Elements qui définissent l'apparence du contenu. Par exemple, les center elements obsolètes big et font
  • Elements HTML sémantiques : éléments qui contiennent du sens et de la structure. ex header, nav, figure.

Elements HTML semantique

  • Element d'en-tête (Header) : utilisé pour définir l'en-tête d'un doc ou d'une section
  • Element principal (main) : utilisé pour contenir le contenu principal de la page Web.
  • Élément de section (section) : utilisé pour diviser le contenu en sections plus petites.
  • Élément Section de navigation (nav) : représente une section avec des liens de navigation.
  • Élément de figure : utilisé pour contenir des illustrations et des diagrammes
  • Élément d'emphase (em) : marque le texte qui a une emphase de stress.
  • Élément Texte idiomatique (i) : utilisé pour mettre en évidence une voix ou une humeur alternative, des termes idiomatiques d'une autre langue, des termes techniques et des pensées.
  • Élément Forte Importance (strong) : marque le texte qui a une forte importance
  • Élément « Attirer l'attention sur (b) : utilisé pour attirer l'attention sur du texte sans importance pour le sens du contenu. Il est couramment utilisé pour mettre en évidence des mots-clés dans les résumés ou des noms de produits dans les avis.
  • Élément Liste de description (dl) : utilisé pour représenter une liste de regroupements de termes-descriptions
  • Description Élément Terme (dt) : utilisé pour représenter le terme en cours de définition
  • Élément Détails de description (dd) : utilisé pour représenter la description du terme
  • Élément Citation en bloc (blockquote) : utilisé pour représenter une section citée à partir d'une autre source. Cet élément possède un cite attribut. La valeur de cet cite attribut est l'URL de la source.
  • Élément Citation (cite) : permet d'attribuer visuellement la source de l'ouvrage référencé. Il marque le titre de la référence.
  • Élément Citation en ligne (q) : utilisé pour représenter une courte citation en ligne.
  • Élément Abréviation (abbr) : utilisé pour représenter une abréviation ou un acronyme. Pour aider les utilisateurs à comprendre ce qu'est l'abréviation ou l'acronyme, vous pouvez afficher sa forme complète, une description lisible par l'utilisateur, avec l'title attribut.
  • Élément Adresse de contact (address) : utilisé pour représenter les informations de contact.
  • Élément (Date) Heure (time) : utilisé pour représenter une date et/ou une heure. Cet datetime attribut permet de traduire les dates et les heures dans un format lisible par machine.
  • Attribut Date (datetime) ISO 8601 : utilisé pour représenter les dates et les heures dans un format lisible par machine. Le format standard est YYYY-MM-DDThh:mm:ss la majuscule T servant de séparateur entre la date et l'heure.
  • Élément exposant (sup) : utilisé pour représenter du texte en exposant. Cet sup élément est couramment utilisé pour les exposants, les lettres supérieures et les nombres ordinaux.
  • Élément d'indice (sub) : utilisé pour représenter du texte en indice. Il est couramment utilisé dans les formules chimiques, les notes de bas de page et les indices variables.
  • Élément Inline Code (code) : utilisé pour représenter un fragment de code informatique
  • Élément Texte préformaté (pre) : représente un texte préformaté
  • Élément Annotation non articulée (u) : utilisé pour représenter une étendue de texte en ligne qui doit être rendue d'une manière qui indique qu'elle comporte une annotation non textuelle.
  • Élément Ruby Annotation (ruby) : utilisé pour annoter du texte avec des explications de prononciation ou de sens. Exemple d'utilisation : typographie d'Asie de l'Est.
  • Élément de parenthèse de secours Ruby (rp) : utilisé comme solution de secours pour les navigateurs ne prenant pas en charge l'affichage des annotations Ruby.
  • Élément texte Ruby (rt) : utilisé pour indiquer le texte de l'annotation Ruby. Il est généralement utilisé pour la prononciation ou les détails de traduction dans la typographie d'Asie de l'Est.
  • Élément barré (s) : utilisé pour représenter un contenu qui n'est plus exact ou pertinent.

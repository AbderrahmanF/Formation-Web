Extension Live Server

nous sert de petit serveur local pour consulter nos pages Web car cela simule un environnement web plus proche de la réalité. En développement web il est quasiment indispensable pour un flux de travail fluide car il recharche également la page à chaque modification.

HTML Comments

You can add comments to your html source by using the following syntax

"<!-- Write your comments here -->"

HTML Favicon

A favicon is a small image displayed next to the page title in the browser tab.

To add a favicon add a <link rel="icon" type="image/x-icon" href"lien de l'icone"

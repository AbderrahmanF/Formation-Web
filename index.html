Formation Web HTML/CSS

Bonjour tu souhaite apprendre les langages de programmation web comme le HTML et le CSS alors tu est au bonne endroit dans cette formation je vais t'apprendre étape par étape les différentes notion en développement web avec bien sur mes mots et la façon dont j'ai compris et assimiler les différentes notions afin de pouvoir les réutiliser lorsque j'en ai besoin.

Conseil pour apprendre

Les conseils ne sont pas évident à mettre en place je le conçoit c'est pour çela qu'il vous faut vous discipliner à les appliquer.

  1. Répetition espacée, essayer de pratiquer tout les jours une notion afin de l'assimiler
  2. Prendre le temps de comprendre, prennez vraiment le temps de comprendre et savoir quand et comment utiliser les notions qu'on va aborder et pas juste les survoler et dire qu'on a compris.
  3. tenir un journal d'avancement, Il est important de noter dans un cahier ou un fichier les notions que vous voyez afin que si vous avez oublier les notions aborder lors d'une précédente séance vous vous les rappelez.
  4. Soyez patient l'apprentissage prend du temps il est normal que vous trouviez çela difficile au début mais si vous tenez et que vous restez constant dans vos efforts vous aller voir de réel progrès avec le temps alors ne soyez pas presser surtout pour la partie CSS.

Prerequis : Outils et logiciels à installer

Pour suivre cette formation, vous aurez besoin de quelques outils et logiciels :

  • Visual Studio code (VsCode) Ce logiciel est un IDE (Environnement de développement) Nous verrons par la suite à quoi çela sert.
  • Navigateur Web, comme Google Chrome ou bien Firefox
  • Extension VsCode Live Server, c'est un petit serveur local qui permet de voir nos pages web comme si elles étaient en ligne. Il est très utile en développement car il met à jour la page automatiquement à chaque changement.
